# 学术规范

## 题目规范

若想要给UranusOJ投稿题目者，可以到[题目投稿专域](https://rpminus.qzz.io/d/problems/)进行投稿，可能会被加入主题库。若你的题目被加入主题库，我们可以给你一些奖励，例如改名、tag。

如果你在题目投稿专域中投稿了题目，则默认UranusOJ有权使用。当然你有该题的完整权益。

在主题库中，题目根据题号前缀内容可以分为以下几种：

- 标准、公开的题目：`P`+编号
- 比赛候选题：`Q`+编号（此类题目普通用户是看不到的，此类题目会隐藏，在比赛结束后会加入主题库）
- 站外题：OJ名+题目在OJ中的编号
- 其他

### 题目投稿评审标准

题意清晰，数据不弱，题面正确使用Markdown，且题目为原创

## 比赛规范

UranusOJ官方一共会举办以下类型的一些公开赛：

- 季节赛：分为春季赛（约每年3月）、夏季赛（约每年7月）、秋季赛（约每年10月）、冬季赛（约每年1月），具体时间不定
- 假期赛：分为寒假赛（每年寒假）、暑假赛（每年8月），具体时间不定
- 节日欢乐赛：国庆欢乐赛、中秋欢乐赛等

等等。

当然，普通用户也可以举办公开赛，不过得向管理员提交申请
